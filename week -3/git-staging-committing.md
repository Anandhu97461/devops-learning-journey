# Git Staging and Committing Changes

## ✍️ Steps:
1. Modify files
2. Stage changes
3. Commit snapshot

## 📥 Staging:
```bash
git add file1
git add "*.sh"

## 🧾 Committing:
```bash
git commit -m "Added script files"

## ❌ Undo Stage:
```bash
git reset file1

## 🗑 Remove File:
```bash
git rm file3
git commit -m "Removed buggy file"
